# An Evidence-Based Study of Multiregional Cortical Contributions to Finger Movement Classification Using ECoG
Files used to process and analyze data from the paper "An Evidence-Based Study of Multiregional Cortical Contributions to Finger Movement Classification Using ECoG"

Abstract—Brain–computer interfaces rely heavily on signals from the sensorimotor cortex, although evidence suggests that other cortical areas may provide complementary information. 
In this work, we analyzed electrocorticographic recordings from finger flexion tasks to compare classification performance using electrodes restricted to the sensorimotor cortex versus extended cortical coverage. 
Mutual information was applied for feature selection, and a linear conditional random fields model was trained and evaluated with cross-validation. 
Performance was quantified using accuracy, balanced accuracy, and Cohen’s kappa. 
The analysis revealed that including electrodes from non–sensorimotor areas, particularly frontal and parietal regions, improved ac-curacy and kappa scores, while balanced accuracy was mostly determined 
by the pro-portion of sensorimotor electrodes. 
Moreover, high–gamma activity provided more reliable features than low-frequency components, supporting its relevance for motor decoding. 
These findings indicate that motor information is not confined to the sensorimotor cortex and that incorporating signals from multiple cortical areas can enhance decoding precision. 
This study provides evidence supporting a multiregional approach to electrode placement in Brain–computer interface systems.

Index Terms— Accuracy, Brain-computer interfaces, Cohen’s kappa, Conditional random fields, electrocorticography, Mutual information, Primary motor cortex.
